# 🥔 Potato Disease Detection using Deep Learning

An AI-powered web application that detects potato leaf diseases using image classification. Built with **Streamlit** and **TensorFlow**, this project helps farmers and researchers quickly identify plant health and take preventive measures.

## 📌 Features

* 📤 Upload potato leaf images
* 🧠 AI-based disease prediction
* 📊 Confidence score visualization using interactive charts
* 🌿 Disease description and treatment suggestions
* ⚡ Fast and user-friendly interface

---

## 🧠 Model Information

* Deep Learning model trained using **TensorFlow/Keras**
* Image classification into 3 categories:

  * Early Blight
  * Late Blight
  * Healthy

⚠️ **Note:** The trained model (`.h5` file) is not included in this repository due to size limitations.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Potato_disease_detection.git
cd Potato_disease_detection
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the application

```
streamlit run app.py
```
